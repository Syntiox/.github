### 🛠 What We Build
We focus on projects that push the boundaries of performance and intelligence:

- **🧠 Artificial Intelligence:** Fine-tuning LLMs (Large Language Models) and developing custom tokenizers for regional languages like Sinhala.
- **⚡ High-Performance Systems:** Building multi-threaded networking tools and scanners using **Rust**.
- **🛡️ Security Research:** conducting ethical hacking research, vulnerability analysis, and developing defensive tools.
- **🤖 Automation:** Creating intelligent bots and APIs to simplify digital workflows.

---

### 🤝 How to Contribute

We welcome developers, researchers, and tech enthusiasts! Here is how you can get involved:

[![](https://img.shields.io/badge/Follow-Syntiox-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Syntiox)

1.  **Explore:** Check out our [Repositories](https://github.com/orgs/Syntiox/repositories).
2.  **Fork:** Click the **Fork** button on any project you find interesting.
3.  **Code:** Fix a bug, add a feature, or improve documentation.
4.  **PR:** Submit a Pull Request, and our team will review it!

> *"Open source is not about code, it's about community."*
